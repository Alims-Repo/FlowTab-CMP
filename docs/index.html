FlowTab

A framework-agnostic bottom navigation bar for Jetpack Compose and Compose Multiplatform with smooth animations and customizable styling.

View on GitHub Get Started

Key Features

Framework Agnostic

Works with any navigation solution including Navigation Compose, Decompose, Voyager, PreCompose, Appyx, or plain state management.

Smooth Animations

Built-in smooth transitions, scale effects, and fluid search bar expansion with customizable timing.

Expandable Search

Built-in expandable search bar with real-time query callbacks and search submission handling.

Glassmorphism Effects

Optional blur effects powered by Haze library for modern, premium visual appearance.

Badge Support

Display notification counts or dot indicators with customizable colors and styling.

Custom Indicators

Choose from Ripple, Dot, or Line selection indicators to match your design system.

Lightweight

Zero navigation dependencies, minimal overhead, optimized for performance and memory efficiency.

Multiplatform

Full support for Android and iOS (iosArm64, iosX64, iosSimulatorArm64) with shared codebase.

Highly Customizable

Extensive styling options including colors, heights, corner radius, elevations, and animations.

Installation

// Add to libs.versions.toml
[versions]
flowtab-cmp = "0.5.1-beta"

[libraries]
flowtab-cmp = { module = "io.github.alims-repo:flowtab-cmp", version.ref = "flowtab-cmp" }

// In your module's build.gradle.kts
dependencies {
    implementation(libs.flowtab.cmp)
}
dependencies {
    implementation 'io.github.alims-repo:flowtab-cmp:0.5.1-beta'
}

Quick Start

@Composable
fun MyApp() {
    var selectedScreen by remember { mutableStateOf("home") }

    val navItems = remember {
        listOf(
            NavItem(
                id = "home",
                label = "Home",
                icon = Icons.Outlined.Home,
                selectedIcon = Icons.Filled.Home
            ),
            NavItem(
                id = "search",
                label = "Search",
                icon = Icons.Default.Search,
                type = NavItemType.Search
            ),
            NavItem(
                id = "profile",
                label = "Profile",
                icon = Icons.Outlined.Person,
                selectedIcon = Icons.Filled.Person,
                badge = BadgeData(count = 3)
            )
        )
    }

    Scaffold(
        bottomBar = {
            BottomNavigation(
                items = navItems,
                selectedId = selectedScreen,
                onItemSelected = { item -> selectedScreen = item.id }
            )
        }
    ) { padding ->
        // Your screen content
    }
}

Selection Indicators

Choose from three indicator styles to match your design system:

━━

Ripple

Full-width background highlight. Bold and high-contrast design.

●

Dot

Small circular indicator. Minimal and modern style.

▬

Line

Horizontal line indicator. Material Design 3 style.

// Ripple Indicator
BottomNavigation(
    config = NavConfig(
        navIndicator = NavIndicator.Ripple(
            color = MaterialTheme.colorScheme.primaryContainer
        )
    )
)

// Dot Indicator
BottomNavigation(
    config = NavConfig(
        navIndicator = NavIndicator.Dot(
            size = 8.dp,
            color = MaterialTheme.colorScheme.primary
        )
    )
)

// Line Indicator
BottomNavigation(
    config = NavConfig(
        navIndicator = NavIndicator.Line(
            height = 3.dp,
            width = 40.dp,
            color = MaterialTheme.colorScheme.primary
        )
    )
)

Configuration Options

Parameter Type Default Description
height Dp 60.dp Height of the navigation bar
cornerRadius Dp 60.dp Corner radius for rounded edges
maxWidth Dp 460.dp Maximum width (useful for tablets)
iconsSize Dp 20.dp Size of navigation icons
animationDuration Int 250 Animation duration in milliseconds
enableBlur Boolean true Enable glassmorphism blur effect
blurIntensity Float 0.95f Blur intensity (0.0 to 1.0)
showLabels Boolean true Show text labels below icons
showBorder Boolean true Show border around navigation bar
elevation Dp 0.dp Shadow elevation
navIndicator NavIndicator Dot() Selection indicator style
